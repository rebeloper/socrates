# Socrates

Socrates is a plugin for [Claude Code](https://claude.com/claude-code) — a collection of skills for the CLI.

This repo is the public face of the project. Development happens in a private companion repo, `socrates-lab`, where skills are built and tested before release here.

## Installing

Add the Socrates marketplace to Claude Code and install the plugin from it, then enable skills as needed.

## Status

Early days — skills are still being added.
